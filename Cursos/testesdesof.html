Testes unitários

Focado em testar unidades individuais de código, como funções, métodos ou classes.
Escritos pelo os próprios desenvolvedores para validar o comportamento esperado de partes específicas do código.
podem ser automatizado e executados frequentemente durante o desenvolvimento.

Testes de integração

Verifica se os defeitos os módulos ou competentes de um sistema funcionam corretamente em conjunto.
Testam a interação entre diferentes espaços do software para garantir que não haja problemas de integração.
